HTML Overview

Introduction

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page.

Basics

  • Elements & Tags: The building blocks of HTML.
  • Attributes: Provide additional information about an element.
  • Comments:
  • Doctype Declaration: Specifies the document type and version.

HTML Document Structure

  • head: Contains meta-information and links to scripts & stylesheets.
  • body: Contains the visible content of the web page.

Common Elements

  • Heading Elements: h1 to h6
  • Paragraph: p
  • Links: a
  • Images: img
  • Lists: ul, ol, li
  • Forms: form, input, textarea, button

HTML5 Features

  • Semantic Elements: article, aside, details, figcaption, figure, footer, header, main, mark, nav, section, summary, time.
  • Graphics: canvas, svg.
  • Media: audio, video.
  • Input Types: date, datetime-local, month, week, time, number, range, search.
